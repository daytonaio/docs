Install Daytona via the Homebrew package manager using the following commands:

```shell
brew tap daytonaio/tap
brew install daytona
```

#### Uninstall Daytona

<Aside type="caution">
This procedure is destructive and irreversible.
Ensure you have a backup of any Workspace data and relevant configuration before proceeding.
</Aside>

<br />

Run the following command to uninstall Daytona:

```shell
brew uninstall daytona
```

#### Update Daytona

Run the following commands to update Daytona:

```shell
brew update
brew upgrade daytona
```
