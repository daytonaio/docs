The `daytona` CLI provides access to all of Daytona's core features.
You can access help for each command using the `--help`/`-h` flag.

## Available Commands

### `autocomplete`
Install a completion script for your shell enviornment.

### `code`
Open a workspace in your preferred IDE

### `create`
Create a workspace

### `delete`
Delete a workspace

### `forward`
Forward a port from a project to your local machine

### `git-providers`
Lists your registered Git providers

### `help`
Provides additional help.

### `ide`
Choose the default IDE

### `info`
Show workspace info

### `list`
List workspaces

### `profile`
Manage profiles

### `purge`
Purges all Daytona data from the current device

### `server`
Start the server process

### `ssh`
SSH into a project using the terminal

### `start`
Start a workspace

### `stop`
Stop a workspace

### `use`
Set the active profile.

### `version`
Displays the currently installed version of Daytona.

### `whoami`
Displays the currently active profile.
