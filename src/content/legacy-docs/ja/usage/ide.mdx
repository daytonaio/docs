Daytonaは、デフォルトのIDEを使用して[ワークスペース](/docs/ja/usage/workspaces#open-an-existing-workspace)に接続するオプションを提供します。

標準で、Daytonaは以下のIDEでワークスペースを開くように設定できます：

- **Visual Studio Code (VSCode)**

  - [デスクトップ (Linux, macOS, Windows)](#vs-code)
  - [ブラウザ](#vs-code-browser)

- **JetBrains IDE**

  - [CLion](#jetbrains-clion)
  - [GoLand](#jetbrains-goland)
  - [IntelliJ IDEA Ultimate](#jetbrains-intellij-idea-ultimate)
  - [PhpStorm](#jetbrains-phpstorm)
  - [PyCharm Professional](#jetbrains-pycharm-professional)
  - [Rider](#jetbrains-rider)
  - [RubyMine](#jetbrains-rubymine)
  - [WebStorm](#jetbrains-webstorm)
  - [Fleet](#jetbrains-fleet)

- [Visual Studio Code Insiders](#vs-code-insiders)

- [VSCodium](#vscodium)

- [Jupyter](#jupyter)

- [Zed](#zed)

- [Cursor](#cursor)

- [Windsurf](#windsurf)

- [Terminal SSH](#terminal-ssh)

## デフォルトIDEの設定

Daytonaで使用するデフォルトIDEを設定できます。

1. 以下のコマンドを実行してデフォルトIDEを設定します：

```shell
daytona ide
```

このコマンドを実行すると、Daytonaは利用可能なIDEのリストを表示し、デフォルトの選択肢を選ぶことができます。

```text
Choose Your Default IDE

VS Code
VS Code - Browser
Terminal SSH
CLion
GoLand
...
```

2. 表示されたリストから希望するIDEを選択します。選択すると、DaytonaはそれをデフォルトIDEとして設定し、確認メッセージを表示します。

```text
Default IDE:  VS Code
```

これで`daytona code`コマンドを使用して、Workspaceに既に接続されているデフォルトIDEを自動的に開くことができます。

## VS Code

DaytonaはVisual Studio Code (VSCode) IDEを使用してワークスペースに接続するオプションを提供します。

1. 以下のコマンドを実行して、デフォルトIDEをVS Codeに設定します：

```shell
daytona ide
```

このコマンドを実行すると、Daytonaは利用可能なIDEのリストを表示し、デフォルトの選択肢として`VS Code`を選択できます。

```text
Choose Your Default IDE

| VS Code
```

2. 表示されたリストから**`VS Code`**を選択します。選択すると、DaytonaはそれをデフォルトのIDEとして設定し、確認メッセージを表示します。

```text
Default IDE:  VS Code
```

これで`daytona code`コマンドを使用して、ワークスペースに既に接続されたVisual Studio Code IDEを自動的に開くことができます。
