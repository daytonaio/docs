DaytonaはApache License 2.0の下でライセンスされたオープンソースの開発環境マネージャー（DEM）です。
Daytonaを使用すると、Workspaceを管理およびデプロイできます。Workspaceは標準のOCIコンテナに基づく再現可能な開発環境で、Dev Container標準の組み込みサポートを提供します。
Daytonaのアーキテクチャは、Dockerfiles、Docker Compose、Nix、DevfileなどのOther設定標準に基づいてWorkspaceを構築する将来の可能性を提供します。

プロジェクトのリモートGitリポジトリの設定を活用することで、DaytonaはWorkspaceを構築し、選択したプラットフォームにワークスペースをプロビジョニングします。
プロビジョニングが完了すると、Visual Studio CodeなどのDaytonaがサポートするIDEを使用してプロジェクトの開発とテストを行うことができます。

Daytonaの機能は、Linux、macOS、Windowsシステム上で動作するコマンドラインツールを通じて公開され、x86-64とAArch64の両方のアーキテクチャをサポートしています。

## 機能

* __セキュリティ__

    Daytonaはクライアントマシンとリモートマシン間にセキュアなVPN接続を作成します。
    リモートマシン上のすべてのポートに、手動でのポートフォワーディングを必要とせずに安全にアクセスできます。

* __Visual Studio CodeとJetBrainsのサポート__

    DaytonaはVisual Studio CodeとJetBrainsのIDE製品群の両方をサポートしており、すべてがローカルにあるかのような感覚でプロジェクトを簡単に開発できます。

* __GitHub、GitLab、Bitbucket、Giteaとの連携__

    Daytonaは、お好みのSCMプラットフォームからリポジトリを取得してDEを作成できます。ワークスペース内でGit操作を実行でき、コンテキストの切り替えなしに作業をプッシュできます。

* __マルチプロジェクトワークスペースのサポート__

    Daytonaは複数のプロジェクトを含むワークスペースを作成できます。
    マイクロサービスや複数のリポジトリに分割された大規模プロジェクトを、単一のワークスペースを使用して作業できます。

* __リバースプロキシサポート__

    Daytonaはリバースプロキシを統合しており、パブリックまたは制限されたネットワーク上でワークスペースにアクセスできます。

* __拡張可能なコア__

    DaytonaはGoで開発されたプラグインをサポートします。
    サードパーティのプロバイダーをDaytonaに追加でき、コア機能への拡張も可能です。
